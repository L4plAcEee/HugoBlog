从 2024 年开始试运行维护自己的个人博客，首先是使用了 Vitepress，然后再到 Hexo，最后兜兜转转来到了 Hugo，这应该是我的最终选择了。  

Hugo 是一个基于 Go 语言 的静态网站生成器，以构建速度极快、结构清晰、工程化程度高而著称，在技术博客与文档站点领域非常受欢迎，这便是我选择的理由，之前使用 Hexo 时，明显能感受到构建速度的缓慢，希望Hugo可以解决这一现象。  

关于安装，直接按照教程来就好，值得注意的是，Config.toml 在新版本变为了 Hugo.toml。

关于主题这一方面，我本人倒是对开发自己的个人主题没什么欲望，直接使用现成的更好。我个人是比较喜欢简约但不简单的主题的，所以很快便锁定了这个 Github 主题，非常推荐。
